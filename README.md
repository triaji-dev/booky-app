# Booky

A modern library management system built with React, TypeScript, and Tailwind CSS.

## Tech Stack

- React 19 + TypeScript
- Tailwind CSS + shadcn/ui
- Redux Toolkit + TanStack Query
- React Router v7
- Axios, Day.js, Sonner, Framer Motion
- Vite

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Features

- **Authentication** — Login, register, JWT session management
- **Book Browsing** — Search, filter, sort, grid/list view
- **Loan System** — Borrow, return, track status, overdue alerts
- **Reviews** — Star ratings, comments, CRUD
- **Shopping Cart** — Batch borrowing with stock checking
- **User Profile** — Edit profile, reading stats, loan history
